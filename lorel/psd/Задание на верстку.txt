Требуется сверстать landing по ссылке

1. Шрифты (со слов дизайнера)
   1. Основной шрифт Open Sans
   2. Шрифт для баннера http://www.fontpalace.com/font-download/Currency+Regular/
   3. Шрифт для цифр в countdown https://www.fontsquirrel.com/fonts/roboto
1. Сайт - landing. Одни страница, при клике по ссылкам идет прокрутка (нативная) к соответствующему разделу landing page.
2. Верстка должна адаптироваться на мобильных устройствах. Дизайн сейчас рисует отдельные картинки для подстановки в адпатированную верстку.
3. Верстка html5, все css должны быть в одном файле.
4. Корректное и максимально возможно унифицированное отображение в браузерах: Chrom Latest Version, FireFox Latest Version, Internet Explorer 8+, Safari OS X 9+


Требования к элементам верстки
1. Справа вверху bullets для навигации. При клике по bullet идет прокрутка на соответствующий раздел страницы. Bullets задаем редактирую html код и активный/наведенный мышкой bullet показывает название (как в дизайне - Home). При прокрутке страницы bullets фиксированные.
2. Countdown вверху слева. В режиме реального времени происходит отсчет вниз от даты, которая вручную указана в html (javascript). При достижении нуля или отрицательного значения, везде показывает нули.
3. Справа по середине кнопка Chat now. Сделать саму фразу Chat now текстом, что бы можно было редактировать.
4. Все картинки сделать не спрайтами а отдельными картинками. Должны лежать в папке /img/. С целью что бы клиент вручную поменял эти картинки на настоящие. В дизайне указал placeholders.
5. Блок видео по середине страницы. Вставляются youtube видео. При нажатии на стрелки карусель.
